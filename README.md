## Git 기본 로직
### git 초기 설정
```
$ git config --global --list
$ git config --global user.name "edujustin"
$ git config --global user.email "edujustin.hphk@gmail.com"
```

### git - github 연결
```
$ git remote add origin [remote repo URL]
```

### git add, commit, push
```
$ git add [파일/폴더명]
$ git commit -m "커밋 메시지"
$ git push origin master
```
